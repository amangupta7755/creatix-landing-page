# Creatix Landing Page 🚀

A modern responsive landing page built using HTML, CSS, and JavaScript.

## 🔥 Features

* Fully Responsive Design (Mobile, Tablet, Desktop)
* Interactive Navigation Menu
* Modal Video Player (YouTube Embed)
* Smooth Scroll Animations
* Clean UI with CSS Variables
* Mobile-Friendly Layout

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)

## 🎯 How It Works

* Click on "Watch Video" → Opens modal
* Video plays using iframe (YouTube embed)
* Click outside / ESC / close button → Modal closes
